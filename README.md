# DriveRadarDMV

DriveRadarDMV is a coming-soon driver intelligence website for Washington, DC, Maryland, and Virginia. The current site is a static MVP preview with placeholder content only.

## Current MVP

- Professional, mobile-friendly homepage
- Planned driver intelligence feature overview
- DMV coverage area overview
- No live APIs, authentication, database, or payments

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Build

```bash
npm run build
```

## Deploy to Vercel

Push the repository to GitHub, import it into Vercel, and use the default Next.js settings. No environment variables are required for the current MVP.
